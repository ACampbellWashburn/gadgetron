The Gadgetron is an Open Source framework for medical image reconstruction. It has been developed at the National Heart, Lung, and Blood Institute, NIH, Bethesda, MD, USA and at the Department of Computer Science and Department of Clinical Medicine, Aarhus University, Denmark. It is made freely available to the medical image reconstruction community.

The Magnetic Resonance in Medicine (MRM) paper on the Gadgetron is now published. If you use the Gadgetron in a scinetific publication, please cite:

Hansen MS, Sørensen TS. Gadgetron: An Open Source Framework for Medical Image Reconstruction. Magn Reson Med. 2012.

Example demonstrations of the framework can be found on our demo page.
The main portal for access to source code, documentation, discussion groups, etc. is the Sourceforge.net website: http://sourceforge.net/p/gadgetron/.
Source code can be found at: http://sourceforge.net/projects/gadgetron/files/.
You can also checkout the code from the git archive with:

git clone git://git.code.sf.net/p/gadgetron/gadgetron
